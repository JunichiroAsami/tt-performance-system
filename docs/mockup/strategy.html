🎾 サーブの作戦

序盤

フォア前に短い下回転

→ 3球目攻撃のチャンス

中盤

バックにナックルと逆回転

→ 相手を迷わせる

終盤

コースを変える、ロングも混ぜる

→ 攻める姿勢を見せる

🏸 レシーブの作戦

短いサーブ

ストップとフリックを半々

→ 先手を取る

長いサーブ

迷わずドライブで攻撃

→ バックサイド狙い

困ったら

深くツッツキで返す

→ ミスを避ける

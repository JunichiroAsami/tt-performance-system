## 1. 概要

### 1.1 目的
卓球パフォーマンス分析システムをWebアプリケーション化し、浅見選手がブラウザから簡単に動画分析・戦略確認・練習計画閲覧ができるようにする。

### 1.2 ターゲットユーザー
- **主要ユーザー**: 浅見江里佳選手（高校生）
- **副次ユーザー**: コーチ、保護者

### 1.3 技術スタック
- **フロントエンド**: React 19 + TypeScript + Tailwind CSS 4
- **UIライブラリ**: shadcn/ui, Lucide React
- **ビルドツール**: Vite
- **デプロイ**: Manus Web Static (Phase 1), Web DB User (Phase 2+)

## 3. 機能要件

### 3.1 戦略チェックリスト機能（CHECKLIST） - 実装済み

| 機能ID | 機能名 | 説明 |
|:---|:---|:---|
| CHECK-01 | 必勝ポイント表示 | 相手に勝つための3つの重要ポイントを優先度順に表示 |
| CHECK-02 | 場面別チェックリスト | サーブ、レシーブ、ラリーごとのチェックリスト |
| CHECK-03 | 達成率表示 | チェック項目の消化率をプログレスバーで表示 |
| CHECK-04 | メンタル・ルール確認 | 「絶対に守ること」と「緊張したときの対処法」を表示 |
| CHECK-05 | 相手情報表示 | 相手の強み・弱点と攻略の結論を表示 |
| CHECK-06 | リセット機能 | 次のセットのためにチェック状態をリセット |

### 3.2 認証機能（AUTH） - Phase 2予定

| 機能ID | 機能名 | 説明 |
|:---|:---|:---|
| AUTH-01 | ログイン | Manus OAuthでログイン |
| AUTH-02 | ログアウト | セッション終了 |

### 3.3 分析機能（ANALYZE） - Phase 2予定

| 機能ID | 機能名 | 説明 |
|:---|:---|:---|
| ANALYZE-01 | 自己分析 | 技術評価、得点/失点パターン、改善提案 |
| ANALYZE-02 | 相手分析 | 相手の強み、弱点、対策 |
| ANALYZE-03 | 定量分析 | 姿勢推定、フットワーク分析（CV連携） |

## 4. UI/UXデザイン（戦略チェックリスト）

### 4.1 デザインコンセプト
**"Neumorphism + Sports Tech"**
- **Calm & Focus**: オフホワイト背景で目に優しく、集中力を阻害しない
- **Tactile**: 物理ボタンのような操作感で誤操作を防止
- **Energy**: オレンジをアクセントカラーに使用し、闘志を鼓舞

### 4.2 配色
- **Base**: #F0F2F5 (Off White)
- **Primary**: #FF6B00 (Ping Pong Orange)
- **Accent**: #1A237E (Deep Navy)
- **Success**: #00C853 (Green)

### 4.3 フォント
- **Headings**: Chakra Petch (Google Fonts) - スポーティー・未来的
- **Body**: Roboto (Google Fonts) - 高い可読性
